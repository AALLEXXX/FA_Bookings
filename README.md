Проектная разработка сервиса "Бронирование отелей"

Стек:
- Fast API
- Python 3.11
- SQLAlchemy
- PostgreSQL
- Redis
- Celery
- Pytest
- Poetry
- SQLAdmin

Что было реализовано:
- регистрация пользователей
- забронировать отель
- отменить бронь
- посмотреть все брони
- посмотреть информацию своего аккаунта
- админ панель SQLAdmin
